🛒 Super Store Sales Dashboard

📊 Project Overview

This project presents an interactive Sales Dashboard built using Power BI to analyze and visualize sales performance of a superstore dataset. The dashboard provides insights into sales, profit, quantity, customer segments, and regional performance.

---

🎯 Objectives

- Analyze overall sales, profit, and quantity
- Identify top-performing categories and sub-categories
- Understand customer segments and payment modes
- Track monthly sales and profit trends (YoY)
- Visualize regional performance across states

---

📌 Key Insights

- 💰 Total Sales: 1.57M
- 📦 Total Quantity: 22K
- 📈 Total Profit: 175.26K
- 🏆 Highest sales from Consumer segment
- 💳 Majority transactions via Online payment mode
- 🚚 Most used shipping: Standard Class

---

📊 Dashboard Features

- KPI Cards → Sales, Quantity, Profit
- Donut Charts → Segment & Payment Mode distribution
- Line Charts → Monthly Sales & Profit (Year-over-Year)
- Bar Charts → Category & Sub-category analysis
- Map Visualization → State-wise sales & profit
- Filters/Slicers → Region-wise analysis (Central, East, South, West)

---

🛠️ Tools & Technologies

- Power BI – Data Visualization
- Python (Pandas) – Data Cleaning & Preprocessing
- Excel/CSV Dataset – Source Data

---

🧹 Data Cleaning Steps
